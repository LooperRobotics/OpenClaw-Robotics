# OpenClaw Robotics - Robot Control via Instant Messaging

## What is this?

OpenClaw Robotics is an open-source skill for the OpenClaw framework that enables natural robot control through instant messaging platforms like WeChat Enterprise (企业微信), Feishu (飞书), DingTalk (钉钉), and WhatsApp.
